#CabsOnline

CabsOnline is a website that allows users to create bookings, and site administrators to view bookings.
As a user, fill in the fields and click the "create a booking" button to submit your booking.
As an admin, search for a booking reference to view details about that booking, or do an empty search to view all bookings due within 2 hours.

Clicking on CabsOnline logo toggles between create booking and admin pages.
